# 🚀 Complete Git & GitHub Guide for Beginners

This guide is designed for **absolute beginners**.
No prior knowledge of Git, GitHub, or VS Code is required.

Follow the steps in order, and you’ll be able to upload and manage your code on GitHub easily.


---

## 🧑‍💻 Step-by-Step Guide

Follow these steps one by one:

### 1️⃣ Install VS Code

First, install Visual Studio Code on your computer.
👉 Follow: [Install VS Code](install-vscode.md)

---

### 2️⃣ Install Git

Next, install Git (Git Bash) which is required to use Git commands.
👉 Follow: [Install Git](install-git.md)

---

### 3️⃣ Create GitHub Account

Now, create your GitHub account if you don’t have one.
👉 Follow: [Create GitHub Account](github-account.md)

---

### 4️⃣ Create Your First Repository

After that, create your first repository on GitHub.
👉 Follow: [Create First Repository](first-repo.md)

---

### 5️⃣ Upload Your Code (First Time)

Now, upload your project/code to GitHub for the first time.
👉 Follow: [Upload Code](upload-code.md)

---

### 6️⃣ Update Your Code

Whenever you make changes, update your code using Git.
👉 Follow: [Update Code](update-code.md)
