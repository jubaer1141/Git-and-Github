# 🧩 Install Visual Studio Code (VS Code)

This guide will help you install **VS Code** on your computer.

---

## 📥 Step 1: Download VS Code

1. Go to the official website:
   👉 https://code.visualstudio.com/

2. Click on **Download for Windows**

---

## ⚙️ Step 2: Install VS Code

1. Open the downloaded file
2. Click **Next**
3. Accept the agreement
4. Keep clicking **Next** until you see **Install**
5. Click **Install**
6. After installation, click **Finish**

---

## ✅ Step 3: Open VS Code

* Open **VS Code** from your desktop or start menu

---

## 🔍 Step 4: Verify Installation

If VS Code opens successfully, then ✅ installation is complete.

---

## 💡 Optional (Recommended)

Install useful extensions:

1. Open VS Code
2. Go to **Extensions (left sidebar)**
3. Search and install:

   * C/C++
   * Code Runner
   * GitLens

---

## 🎉 Done!

You have successfully installed VS Code.
