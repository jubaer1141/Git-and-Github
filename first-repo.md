# 📦 Create Your First GitHub Repository

This guide will help you create your **first repository on GitHub**, where your code will be stored.

---

## 🌐 Step 1: Go to GitHub

1. Open your browser
2. Go to:
   👉 https://github.com/
3. Login to your account

---

## ➕ Step 2: Create New Repository

1. Click the **"+" (plus icon)** at the top-right corner
2. Click **"New repository"**

---

## 📝 Step 3: Fill Repository Details

1. **Repository Name**

   * Example: `my-first-project`

2. **Description (Optional)**

   * Example: My first GitHub project

3. Choose:

   * ✅ Public (recommended for beginners)
   * OR Private

4. (Optional but recommended)

   * ✔️ Check **"Add a README file"**

---

## 🚀 Step 4: Create Repository

* Click **"Create repository"**

---

## ✅ Step 5: Repository Created

You will now see your repository page.

Your repository is ready to use 🎉

---

## 🔗 Important

You will use this repository to upload your code in the next step.

---

## 🎉 Done!

You have successfully created your first GitHub repository.
