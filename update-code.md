# 🔄 Update Your Code on GitHub

This guide will help you **update your project/code on GitHub** after making changes.

---

## 📁 Step 1: Open Your Project Folder

* Go to your project folder
* Example: `Desktop/my-project`

---

## 💻 Step 2: Open Git Bash

* Right-click inside the folder
* Click **"Git Bash Here"**

---

## ✏️ Step 3: Make Changes

* Edit your files (add, delete, or update code)
* Save your changes

---

## ➕ Step 4: Add Changes

```bash id="m8n2x4"
git add .
```

---

## 💾 Step 5: Commit Changes

```bash id="p3k9d1"
git commit -m "update your message"
```

Example:

```bash id="x7v2q5"
git commit -m "updated login feature"
```

---

## 🚀 Step 6: Push Changes

```bash id="c4z8t6"
git push
```

---

## 🔍 Step 7: Check on GitHub

* Go to your repository on GitHub
* Refresh the page
* You will see your updated code

---

## 💡 Tips

* Always write clear commit messages
* Example:

  * "fixed bug"
  * "added new feature"
  * "updated UI"

---

## ⚠️ If You Get Error (First Time Only)

If Git asks to set upstream, use:

```bash id="v2n8k1"
git push -u origin main
```

After that, you can use just:

```bash id="j6m3p9"
git push
```

---

## 🎉 Done!

You have successfully updated your code on GitHub.

Keep practicing and your workflow will become faster 🚀
